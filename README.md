# ror-practice
Ruby on Rails Practice Repo

## Prerequisites
- ruby 2.3.7
- rails 3.2.22.1
- rubocop (ruby lint)
- rspec (testing)
- react 16.8
- redux 7.1
- eslint (lint)
- jest (testing)
- enzyme
- docker

## Getting Started
To run the application (install docker first) on development
```
docker-compose build && docker-compose up -d
```
Then open your browser on localhost:3100

## Running the test

### Frontend
```
cd frontend
npm test
```
### Backend
```
cd backend
rspec
```
